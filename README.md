## 📖 Descrição do projeto
Este projeto consiste no desenvolvimento de uma aplicação web utilizando HTML e CSS, com o tema dos times da Série A do Campeonato Brasileiro.

A aplicação apresenta:
- Lista completa dos clubes da Série A
- Escudos e descrições dos times
- Seção de títulos (Brasileirão e Libertadores)
- Área de comentários
- Menu de navegação interno
- Sistema de pesquisa por seleção

---

## 🎯 Funcionalidades
- 🔍 Pesquisa de times (sem uso de JavaScript)
- 📋 Listagem de todos os clubes
- 💬 Área de comentários
- 🏆 Exibição de títulos
- 🎨 Identidade visual com cores dos times

---

## 📱 Responsividade
O site foi desenvolvido para funcionar em diferentes dispositivos:

- 💻 Desktop → layout em duas colunas  
- 📱 Mobile → layout em uma coluna  

Utilizando:
- CSS Grid
- Flexbox
- Media Queries
